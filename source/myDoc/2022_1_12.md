### list(1,2,3)转为list(2,4,6)

```python
l = [1, 2, 3]
# 1. 列表推导式
l = [i*2 for i in l]
# 2. map函数
l = map(lambda x:x*2, l)
# 3. numpy
import numpy as np
l = (np.array(l)*2).tolist()
```
